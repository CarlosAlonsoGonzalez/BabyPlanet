# ProyectoAplicacion

### "BabyPlanet"
Nuestra idea de proyecto es una aplicación enfocada a padres primerizos en los primeros años del bebé, para poder guiarlos y ayudarlos a desempeñar de la manera más óptima la tarea de ser padre.
